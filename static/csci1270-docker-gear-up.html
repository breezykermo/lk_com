What is Docker?

A software toolkit for running containers.

Containers are essentially lightweight virtual machines, providing a way to run arbitrarily complex software without installing anything but Docker on the host machine.

Docker deals with underlying OS (Windows, Mac, Linux) in the host: our software runs on a unified virtual OS.

Installation

Docker Desktop installation instructions:

  • MacOS (x86_64 and arm64)
  • Windows
  • Linux

What is Docker Compose

An orchestration tool.

Orchestrates various containers running together, with open ports, networking, and file system mappings.

Allows us to orchestrate a set of relevant containers (simulating distinct machines) with a single command: docker-compose up

Installation

Docker Compose is included with Docker Desktop.

See this documentation for more information.

What is Jupyter Lab/Notebook?

A web interface used for interactive computing.

Particularly popular in data science and machine learning, and typically used with a Python kernel.

Provides both executable and Markdown cells, which can be interspersed in a notebook.

Installation

Throughout the course, we always run Jupyter within a Docker container as part of a Docker Compose orchestration.

This insulates students from having to install Jupyter or supporting libraries on their host machine, and allows us to configure custom environments (useful in Distributed and Streaming assignments)
